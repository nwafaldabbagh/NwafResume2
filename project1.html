VLAN & Trunking Lab (Cisco Packet Tracer)

Configuring VLANs, trunk ports, and inter-VLAN routing across multiple switches.

Overview

Built a campus-style network topology with access and trunk ports, created multiple VLANs, and verified segmentation and connectivity. Implemented inter‑VLAN routing to enable controlled communication between subnets.

What I Did

  • Designed the topology and assigned switchports to the correct VLANs.
  • Configured 802.1Q trunking on uplinks; set native VLANs and allowed lists.
  • Enabled inter‑VLAN routing using a router‑on‑a‑stick or L3 switch SVI.
  • Tested connectivity with ping and verified MAC address tables.
  • Documented commands and results.

Skills Learned

Network segmentation 802.1Q trunking SVIs & inter‑VLAN routing Cisco CLI Troubleshooting
